# Soundvisualisation

## Loki
- logs are saved in [Loki](routes/jsons/conf.json)

## Usage
1. Install [Node.js](https://nodejs.org/en/download/)

2. Install npm packages it
```
npm install
```
3. Start it
```
npm start
```
4. Goto [http://localhost:3000/](http://localhost:3000/)
